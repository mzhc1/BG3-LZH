# BG3-LZH — Learn Chinese While Playing

> Play Baldur's Gate 3 in English. Learn Mandarin passively.

![GitHub License](https://img.shields.io/github/license/mzhc1/BG3-LZH)

---

## What is this?

BG3-LZH adds **pinyin annotations** to **252 HSK1 words** in BG3's Chinese subtitles. No flashcards. No textbooks. Just play and absorb the language naturally.

Based on [this Chinese localization mod](https://www.nexusmods.com/baldursgate3/mods/2582) on Nexus Mods.

## Installation

**Requirements:** Baldur's Gate 3 — game language must be set to **English**.

1. Download `English.pak` from this repo
2. Navigate to `Baldur's Gate 3/Data/Localization/`
3. Back up the original `English.pak`
4. Replace it with the downloaded file

> No mod manager needed.
